Samoan Local Favorite Foods

Personal Web Application Site Plan

1. Purpose and Audience

The purpose of this website is to present and celebrate some of the most loved traditional Samoan foods. The site will showcase a collection of dishes with descriptions, images, and cultural context formatted in a clean and visually appealing way.

The intended audience includes classmates, instructors, family members, and anyone interested in learning about Samoan cuisine or Pacific Island culture. Content is written simply and clearly so that users who may be unfamiliar with Samoan foods can easily understand and enjoy the site.

2. Dynamic Elements

JavaScript will add interactive features across the site. A separate module will store an array of food objects containing dish names, categories, images, ratings, and descriptions. The Foods page will include a live search bar that filters foods using the filter() array method, plus a category filter (Main Dish, Seafood, Dessert) that uses conditional logic to update which items appear.

The Home page will feature a “Random Food Generator” button that selects and displays a random dish using DOM manipulation. ES Modules will organize the code into multiple functions. These features demonstrate arrays, objects, array methods, event handling, DOM interaction, and conditional branching.

5. Content

Home Page Content

Welcome to “Samoan Local Favorite Foods,” a simple and informative site highlighting several traditional foods from the islands of Samoa. From coconut-rich dishes to sweet island treats, this site shares a collection of well-loved meals that hold cultural and personal significance. Use the Random Food Generator to discover a dish or explore the full food list.

Foods Page Content

This page displays a grid of Samoan foods with category filters and a search bar. Each card includes an image, the food name, category, and a brief description. Users can browse, filter, and learn more about each dish.

Food Data Items

  • Palusami: Taro leaves baked in coconut cream (Main Dish).
  • Oka i’a: Fresh raw fish marinated in lemon and coconut milk (Seafood).
  • Sapasui: Samoan chop suey with noodles and savory sauce (Main Dish).
  • Panikeke: Sweet round Samoan pancakes (Dessert).
  • Keke Pua’a: Steamed pork buns (Main Dish).
  • Faiai Eleni: Mackerel baked in coconut cream (Seafood).
  • Fa’ausi: Taro topped with caramel coconut sauce (Dessert).
  • Talo (Taro): A staple Samoan starch served with meals (Main Dish).
